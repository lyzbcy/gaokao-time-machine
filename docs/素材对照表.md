## 替换素材的方法

所有图片文件名是固定的（见上表"项目内文件"列），直接用新图覆盖同名文件即可，无需改动任何 JS/CSS。
建议尺寸：标签表情 240×240 或更大正方形；透明 PNG 优先。

## 复制脚本

素材通过 `copy-assets.js` 从素材池复制而来：
```
node copy-assets.js
```
该脚本用 Node 而非 PowerShell，规避了 Windows 中文路径编码问题（PowerShell 5.x 读 ps1 文件按 GBK 解码，会乱码）。
