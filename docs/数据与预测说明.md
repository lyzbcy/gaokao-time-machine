## 一·补、位次锚定换算（核心机制）

### 为什么用位次而不是分数
每年高考试题难度不同，同一分数的"含金量"年年变（例如 2020 年 600 分和 2024 年 600 分能上的学校可能差一档）。但**全省排名（位次）相对稳定**——第 1 万名就是第 1 万名。所以本网页用「**位次锚定**」做跨年份换算：

```
你当年分数 → 当年一分一段表 → 当年全省排名
              ↓（同位次）
2026 一分一段表 → 2026 等位分
              ↓
用 2026 等位分判定能上哪些学校 → 对比当年档次 → 出标签
```

### 一分一段表数据来源（按精度分层，动态校准）

位次换算的精度完全取决于一分一段表的质量。当前数据分层如下：

| 数据源 | 数量 | 说明 |
|--------|------|------|
| **official 真实表** | 34 张 | 从教育在线（eol.cn）抓取的真实一分一段表，覆盖 23 省，含完整逐分点位 |
| **official-extrapolated 锚点扩展** | 87 张 | 基于同省真实表锚点校准扩展到其他年份/科类 |
| **predicted 纯模型** | 327 张 | 无真实数据省份/历史年份，用锚点校准 sigmoid 模型生成 |

合计 448 张表。**真实 + 锚点合计占 27%**，且随抓取持续推进持续提升。

模型公式（锚点校准 sigmoid）：
- 生成 S 型累计分布曲线（高分段稀疏、低分段密集，符合高考实际）
- 有真实锚点时，用 sigmoid 曲线拟合真实点位
- 无真实锚点时，回退到默认校准锚点（以四川2023理科为基准）

### 已抓取真实数据的 23 省（截至 2026-06）

四川、河南、山东、广东、江苏、湖北、湖南、浙江、安徽、重庆、福建、江西、陕西、黑龙江、吉林、北京、上海、**内蒙古、广西、海南、甘肃、青海、天津**（加粗为第二批补全）

> 剩余 8 省（河北、辽宁、山西、云南、贵州、宁夏、新疆、西藏）在 eol.cn 上以**图片形式**发布一分一段表，无法直接抓取 HTML 表格，待后续 OCR 处理。

### 科类口径（分省滚动切换）

各省新高考改革分批落地，一分一段表口径随年份切换：

| 省份类型 | 2019-衔接年前 | 衔接年后 |
|----------|--------------|----------|
| 浙江、山东、北京、上海、天津、海南 | 3+3 总分 | 3+3 总分 |
| 冀苏鄂湘粤闽渝 | 文/理科 | 物理/历史类（2021 起） |

### 爬虫与渐进式精度提升
- **合规边界**：只爬官方源（教育在线 eol.cn 转载各省考试院数据）。不爬商业 AI 预测站，避免侵权。
- **抓取脚本**：`node scripts/fetch-yfyd.js`，Playwright 抓取 HTML 表格，解析逐分点位写入 `rank-tables.json`。
- **重试机制**：eol.cn 国内访问慢且不稳（ERR_NETWORK_CHANGED 常见），脚本内置 3 次指数退避重试 + browser 全局复用。
- **图片格式省份**：8 省发布为图片，HTML 表格抓不到，需走 `scripts/ocr-keypoints.js`（tesseract.js）。

## 二、预测模型说明

### 模型类型
**加权线性回归（Weighted Linear Regression）**（用于学校录取分预测）

### 计算逻辑
1. 取某校某省 2019-2025 的录取最低分序列
2. 对近年数据赋更高权重（指数衰减，base=1.35）：
   - 2025 年权重 = 1.0
   - 2024 年权重 ≈ 0.74
   - 2023 年权重 ≈ 0.55
   - ……
3. 拟合直线 `y = a + b·year`，外推到 2026 年
4. **限幅**：单年波动不超过 ±15 分，避免外推爆炸
5. 数据不足 3 年时退化为线性外推或单点沿用

### 置信度
每个预测附带一个娱乐性"置信度"指标（20%-95%）：
- 样本年份数越多 → 越高
- 历年分数波动越小 → 越高
- **纯娱乐指标，不代表统计学意义的置信区间**

## 三、权重说明（回答初心：分数怎么来的）

网页展示的两部分数据，来源不同：

```
位次换算（等位分）：23省用真实一分一段表，8省用锚点校准模型
                   ↑ 这部分精度较高，是标签判定的核心依据

2026 学校分数线：100% 加权线性回归模型预测
                （基于 2019-2025 代表性估算数据外推）
```

**没有爬取任何第三方 AI 预测网站的数据。** 历史院校分数为基于梯队/省份系数的代表性估算，2026 学校分数完全由自研加权线性回归模型算出。

## 四、盲盒标签系统

标签基于「当年能上的学校档次」与「同分映射到 2026 年能上的档次」的对比生成，具体触发条件见 `初心与使命.md` 的标签分级表。

标签概率分布（显示概率，用于排行榜统计）：
- SSR 天选之子：2%
- SR 血亏战神 / 时代红利：各 8%
- R 幸好考得早 / 捡漏王 / 平平无奇：25% / 15% / 20%
- N 一代不如一代 / 学费通胀侠：12% / 10%
- 隐藏款（平行宇宙 / 时空悖论）：不公开概率，特殊条件触发

> 注：displayProb 是给用户看的显示概率（营销文案），真实抽取由用户输入的条件命中决定，二者不必相等。
