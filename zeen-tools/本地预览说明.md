# 本地预览说明 · 高考分数时光机

## 启动方式

**双击运行：** `zeen-tools\一键启动前端.bat`

脚本会自动：
1. 检测 Node.js 是否安装
2. 释放可能被占用的 8091 端口
3. 启动 `local-preview-server.js`
4. 健康检查首页（等待返回 200）
5. 自动打开浏览器到首页

**命令行启动（等效）：**
```bash
npm run preview:local
# 或
node local-preview-server.js
```

## 停止方式

**双击运行：** `zeen-tools\一键关闭前端.bat`

会按 `local-preview-server.js` 命令行特征筛出 node 进程并停止，同时释放 8091-8095 端口。
（不只是关闭窗口，而是真正结束进程、释放端口。）

## 默认地址

| 入口 | 地址 |
|------|------|
| 首页 | http://127.0.0.1:8091/ |
| 导航页 | http://127.0.0.1:8091/nav |

> 端口 8091 被占用时，服务自动回退到 8092-8095，控制台会打印实际端口。

## 推荐抽查顺序

1. **首页开盒**（`/`）：选省份（如四川）、年份（如 2021）、分数（如 580）→ 点击「开盒」→ 看动画 → 跳转结果页
2. **结果页**（`/#/result`）：标签卡 + 对比表 + 骚话 + 点「生成分享卡片」看 canvas 出图
3. **排行榜**（`/#/ranking`）：切 3 个 tab（稀有度 / 省份 / 内卷）
4. **我的图鉴**（`/#/collection`）：看刚开盒解锁的标签 + 历史记录
5. **导航页**（`/nav`）：切「📱 Mobile」模式，点卡片看手机视口预览

## Mobile 预览说明

导航页右上角有两个按钮：
- **普通**：点击卡片在新标签页直接打开对应 hash 路由
- **📱 Mobile**：点击卡片弹出一个 375×812 的手机外壳窗口（带刘海），内嵌 iframe 加载该路由，模拟手机端真实视口

本项目是面向手机用户的短视频引流网页，建议用 Mobile 模式抽查移动端适配效果。

## 这个预览服务解决了什么问题

1. **中文路径不乱码**：所有 bat 用 `chcp 65001`，响应头带 `charset=utf-8`，项目路径含中文和空格也能正常工作
2. **端口冲突自愈**：启动前先释放旧进程，运行时端口被占自动回退
3. **真正释放端口**：关闭脚本按进程特征精准停止，不会留下僵尸 node 进程
4. **SPA 友好**：所有 hash 路由（`#/`、`#/result` 等）共享同一个 `index.html`，服务端无需特殊处理
5. **双击即用**：不依赖 IDE、不依赖框架热更新，非开发人员也能双击运行

## 常见问题

**Q: 双击 bat 闪退？**
A: 多半是没装 Node.js。到 https://nodejs.org 下载 LTS 版安装。

**Q: 浏览器打开了但显示「无法访问」？**
A: 服务可能还在启动中，看新弹出的命令行窗口是否有「本地预览已启动」字样。若端口回退到 8092，bat 会提示实际地址。

**Q: 想换端口？**
A: 改 `一键启动前端.bat` 里的 `set "LOCAL_PREVIEW_PORT=8091"`，或用 `set LOCAL_PREVIEW_PORT=8080 && node local-preview-server.js` 环境变量覆盖。
