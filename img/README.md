# img 目录 · 美术资源

本目录所有图片来自 `E:\星星布丁\微信表情包`，为自有 IP 素材（星星布丁 / 捞鱼 / 周三涵）。

## 子目录

- `labels/` — 10 个标签（8 常规 + 2 隐藏）对应的表情图
- `ui/`     — 开盒按钮、Logo 等 UI 元素

## 文件清单与映射

详见 `../docs/素材对照表.md`。

## 来源声明

网页底部及分享卡片均标注「🎨 表情包友情出演：星星布丁 · 捞鱼 · 周三涵」，
用于推广个人 IP。

## 替换方式

文件名固定，直接用新图覆盖同名文件即可，无需改代码。
推荐规格：透明 PNG，正方形，≥240×240。

## 重新复制

如需从素材池重新拉取，运行：
```
node copy-assets.js
```
（根目录下的脚本，用 Node 跑以规避 PowerShell 中文路径编码问题。）
